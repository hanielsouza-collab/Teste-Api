# CatalogoFlix API

## Descrição
API REST para catálogo de filmes, permitindo listar, buscar, cadastrar, atualizar e remover filmes.

## Tema
Filmes

## Integrantes
- Haniel Sousa
- Kaique de Souza
- Gabriel Henrique
- Matheus Pereira

## Endpoints

- GET /listar
- GET /buscar/:id
- POST /cadastrar
- PUT /atualizar/:id
- DELETE /remover/:id

## Tecnologias
- Node.js
- Express
- JavaScript
- JSON

## Base de Dados
Array interno simulando um banco de dados com filmes.

## Execução
npm install  
node server.js
